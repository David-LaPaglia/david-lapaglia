Sustainability Analysis Project

Project Overview

This comprehensive analysis explores public discourse and scientific understanding of climate change through advanced text mining techniques. By analyzing diverse data sources, including social media discussions, scientific publications, and news articles, this project aims to uncover patterns in how different communities perceive and discuss climate-related issues.

Unsupervised Learning Methods

Clustering

Clustering is used to uncover insights about unlabeled data, this is called unsupervised learning. It is useful for looking at data with new perspectives.

Association Rule Mining

Association Rule Mining (ARM) is a data mining technique used to discover interesting relationships between variables in large datasets.

Latent Dirichlet Allocation

Latent Dirichlet Allocation (LDA) is a generative statistical model used in natural language processing and text mining to discover topics in a collection of documents.

Project Significance

This analysis contributes to our understanding of climate change communication by:

  • Identifying gaps between scientific consensus and public understanding
  • Revealing how different communities frame and discuss climate issues
  • Tracking the evolution of climate change discourse over time
  • Providing insights for more effective climate communication strategies
