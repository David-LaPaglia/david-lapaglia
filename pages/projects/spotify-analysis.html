Beats and Bytes: Song Popularity Analysis

« Back to Projects

Project Overview

This group project analyzed Spotify data to answer the fundamental question: what makes a song popular? The analysis explored various musical attributes and their correlation with popularity, revealing interesting trends in music over time.

Research Question

The project sought to identify which musical attributes and characteristics most strongly correlate with song popularity on streaming platforms. Additionally, we investigated how these attributes have evolved over time.

Methodology

Our approach included:

  • Data Collection: Gathered comprehensive song data from Spotify's API
  • Feature Analysis: Examined musical attributes such as energy, danceability, tempo, acousticness, and valence
  • Temporal Analysis: Tracked changes in musical preferences and attributes over decades
  • Correlation Study: Identified relationships between specific song characteristics and popularity metrics
  • Visualization: Created informative visualizations to present findings and trends

Key Findings

The analysis revealed several interesting insights:

  • Energy levels in popular songs have decreased over time, reflecting changing listener preferences
  • Specific combinations of musical attributes strongly correlate with higher popularity
  • Genre-specific patterns in what makes songs successful
  • Seasonal variations in song popularity and attribute preferences

Tools & Technologies Used

  • Python data analysis libraries (Pandas, NumPy)
  • Data visualization tools (Matplotlib, Seaborn)
  • Spotify API for data collection
  • Statistical analysis techniques
