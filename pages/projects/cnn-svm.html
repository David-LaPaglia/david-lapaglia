CNN + SVM Classification

« Back to Projects

Project Overview

This project focused on image classification using both Convolutional Neural Networks (CNN) and Support Vector Machines (SVM). The goal was to compare the effectiveness of these two different approaches for classifying traffic light images by color.

Methodology

The project consisted of several key components:

  • Data Collection: Gathered images of red, green, and yellow traffic lights from the internet
  • Data Preprocessing: Normalized and regularized the image data to ensure consistency
  • CNN Model: Developed a convolutional neural network model for image classification
  • SVM Model: Created a Support Vector Machine model from scikit-learn as a "white-box" alternative
  • Comparative Analysis: Evaluated both models based on accuracy, precision, recall, and F1-score

Key Findings

The comparative analysis revealed important insights:

  • Performance differences between CNN and SVM for image classification tasks
  • Trade-offs between model complexity and interpretability
  • Strengths and weaknesses of each approach for specific classification scenarios
  • Optimal preprocessing techniques for improving model performance

Tools & Technologies Used

  • Python (TensorFlow/Keras for CNN, scikit-learn for SVM)
  • Image preprocessing libraries
  • Model evaluation and performance metrics
  • Data visualization tools
