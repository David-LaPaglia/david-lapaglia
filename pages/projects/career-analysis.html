Career Skills Analysis

« Back to Projects

Project Overview

An in-depth analysis of career data, pitted against the demands of academia. This project explores the gap between skills taught in academic programs and those sought after in the job market.

Methodology

This Exploratory Data Analysis project utilized several Python libraries and techniques:

  • Data Collection: Used Python's requests module to scrape job posting data
  • Data Parsing: Employed BeautifulSoup to extract relevant information from HTML
  • Data Analysis: Leveraged PANDAS for data manipulation and analysis
  • Data Visualization: Created visualizations using Seaborn to identify patterns and trends

Key Findings

The analysis revealed several insights about the alignment between academic curriculum and industry requirements:

  • Top skills demanded by employers that are underrepresented in academic programs
  • Correlation between certain skills and salary ranges
  • Regional variations in skill demand
  • Emerging skills that show increasing demand over time

Tools & Technologies Used

  • Python (requests, BeautifulSoup, PANDAS, Seaborn)
  • Data scraping and cleaning techniques
  • Statistical analysis
  • Data visualization
